# Sitio Web Institucional — Instituto Tecnológico Superior de Cintalapa

## Descripción

Este proyecto consiste en el desarrollo de un nuevo sitio web institucional para el Instituto Tecnológico Superior de Cintalapa. El objetivo del sistema es diseñar una plataforma web moderna, clara y funcional que permita presentar información institucional de manera organizada y accesible para estudiantes, docentes, personal administrativo y público en general.

El proyecto surge como parte de las actividades realizadas durante el servicio social, en donde se planteó la necesidad de rediseñar completamente el sitio web institucional con el fin de mejorar la estructura de la información, la accesibilidad del contenido y la experiencia de navegación de los usuarios.

A diferencia del sitio actual, esta propuesta busca desarrollar una nueva plataforma desde cero, evitando problemas estructurales existentes y permitiendo una organización más clara de los contenidos institucionales.

---

## Estado del proyecto

Actualmente el proyecto se encuentra en desarrollo. En esta fase se está trabajando principalmente en el diseño estructural del sitio web, la organización del contenido institucional y la implementación de los componentes principales de la interfaz.

El objetivo es construir una base sólida que permita posteriormente integrar funcionalidades adicionales y mejorar progresivamente la experiencia del usuario.

---

## Tecnologías utilizadas

El desarrollo del sitio web se está realizando utilizando las siguientes tecnologías:

- HTML5
- CSS3
- JavaScript
- Diseño responsivo para distintos dispositivos
- Estructuración modular del contenido

Estas tecnologías permiten construir una plataforma ligera, adaptable y compatible con distintos navegadores y dispositivos.

---

## Objetivo del proyecto

El objetivo principal del proyecto es desarrollar un sitio web institucional que permita difundir información relevante del Instituto Tecnológico Superior de Cintalapa de manera organizada, accesible y visualmente clara.

Entre los propósitos del proyecto se encuentran mejorar la presentación de información institucional, facilitar el acceso a contenidos académicos y administrativos, y proporcionar una estructura de navegación más intuitiva para los usuarios.

---

## Avances actuales

Hasta el momento se han desarrollado algunos de los componentes fundamentales del sitio web, entre los cuales se incluyen:

- Estructura base del sitio web.
- Diseño inicial de la interfaz.
- Organización preliminar de secciones institucionales.
- Implementación de algunos elementos de navegación.

Estos avances forman parte de las primeras etapas del desarrollo del sistema.

## Ejecución del proyecto

Para visualizar el proyecto es necesario realizar los siguientes pasos:

1. Descargar el archivo comprimido disponible en el repositorio.
2. Descomprimir el archivo utilizando un programa como WinRAR o 7-Zip.
3. Abrir el archivo principal del sitio web (index.html) desde un navegador web.
4. Navegar entre las diferentes secciones disponibles dentro del sitio.
